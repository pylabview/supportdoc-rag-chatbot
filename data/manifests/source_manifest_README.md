
# source_manifest.jsonl

Example manifest for the RAG ingestion pipeline.

Each line represents a source document approved for ingestion.

Fields:

snapshot_id — corpus snapshot identifier
source_path — repo path inside Kubernetes website repo
source_url — canonical documentation URL
doc_id — stable identifier
language — document language
license — document license (CC BY 4.0)
attribution — attribution text
allowed — allowlist decision

This file is normally generated by:

    build_manifest.py
